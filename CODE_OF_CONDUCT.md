# Code of Conduct

This project follows the [Rust Code of Conduct](https://www.rust-lang.org/policies/code-of-conduct).

## Short version

- Be civil. Disagree with the code, not the contributor.
- No harassment, no doxxing, no political flame wars in issues / PRs / commit messages.
- English is the working language. Non-native speakers get the
  benefit of the doubt on tone; native speakers don't.

## Enforcement

Report violations privately to the maintainer via
[GitHub Security Advisories](https://github.com/mcaney006/grokmacos-better-version/security/advisories/new)
with `[code-of-conduct]` in the title. Expect a response within 72 hours.

The maintainer may:

1. Edit or delete the offending content.
2. Lock the thread.
3. Temporarily ban the user from the repo.
4. Permanently ban repeat offenders.

Decisions are not negotiable in the comment thread.
